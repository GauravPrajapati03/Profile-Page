# Profile Page

A simple profile page built using HTML, CSS and JavaScript.

## Features

- **Responsive design**: The page adjusts to different screen sizes and devices.
- **Navigation bar**: Links to different sections of the page for easy navigation.
- **Profile section**: Includes an image, name, and description.
- **Skills section**: Lists the skills with icons or text.
- **Projects section**: Displays a list of projects with descriptions or links.
- **Social Media & Contact Info**: Section with links to social media profiles and contact information.
- **Typing Animation**: An animated text effect that simulates typing text on the page.
- **Back-to-Top Button**: A button that appears when the user scrolls down, which smoothly scrolls the page back to the top.
- **Active Navigation Link Color Change**: Navigation links change color when clicked to indicate the active section.
